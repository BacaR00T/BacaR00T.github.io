Main NAS — TrueNAS RAID-Z1

2025-10-09 · ← all projects · /home

My main storage server, designed for reliability and quiet operation. It runs TrueNAS Core bare-metal and manages all my network shares, backups, and media data.

// Hardware

  • Intel i7-3820
  • 16 GB DDR3 RAM
  • Boot: 240 GB SATA SSD
  • Storage: 4 × 5 TB HDDs (RAID-Z1)

// Role

  • TrueNAS Core — main OS
  • RAID-Z1 pool for redundancy
  • SMB shares for home network
  • Snapshot + backup automation

Why this setup?

This build balances storage capacity, data protection, and power efficiency. The i7-3820 handles ZFS compression and checksums easily, while RAID-Z1 provides one-drive fault tolerance with solid throughput.
