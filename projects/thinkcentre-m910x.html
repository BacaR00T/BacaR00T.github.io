ThinkCentre M910x Homelab

2025-10-08 · ← all projects · /home

Small but mighty SFF box running game + services. Target: low power, quiet, easy to maintain.

// Hardware

  • Lenovo ThinkCentre M910x (SFF)
  • CPU: i7-6700
  • RAM: 32 GB
  • Storage: 256 GB NVMe + 2 TB SATA SSD

// Role

  • Host OS: Proxmox
  • Debian VM with CasaOS
  • Crafty panel for Minecraft servers

Screens / Photos

Network & Services

  • CasaOS apps: backups, media tools, monitoring
  • Crafty: start/stop MC servers, backups, logs
  • VM layout: Proxmox → Debian VM (Crafty/CasaOS)

Setup notes

# Proxmox install → create Debian VM
# Basic hardening + updates
sudo apt update && sudo apt -y upgrade

# CasaOS (one-liner)
curl -fsSL https://get.casaos.io | sudo bash

# Crafty (inside Debian VM)
sudo apt -y install default-jre screen
# follow Crafty install docs; expose web UI via reverse proxy or port map

Why this box?

Great perf-per-watt, cheap used, runs multiple MC instances smoothly. NVMe for OS, SATA SSD for worlds/backups.
